Wirnet iFemtoCell LoRaWAN Gateway

The Wirnet iFemtoCell and the Wirnet iFemtoCell-evolution compose Kerlink’s LoRaWAN indoor gateway range

Based on the same Hardware and Software, they only differ by the type of backhaul connection they propose.

The Wirnet iFemtoCell and Wirnet iFemtoCell-evolution range is the ideal LoRaWAN indoor gateway range to support your smart building, smart city or any smart project that requires dedicated deep indoor coverage and/or network densification, providing both unique superior coverage and operational excellence.

This LoRaWAN gateway offers high performance in demanding environments (basement, underground, parking, elevators shafts…) to connect energy meters, parking sensors or equipment actuators.

The Wirnet iFemtoCell range is powerful and robust

These IoT LoRaWAN gateways leverage the successful LoRa RF technical design and the common software of Wirnet i-Series gateways.

In addition, they embed built-in high rejection filters and secured Software architecture to be reliable gateways in the core of your smart application.

The Wirnet iFemtoCell and iFemtoCell-evolution are easy to manage

The Wirnet iFemtoCell and iFemtoCell-evolution are easy to configure, and come with exclusive features for its remote monitoring and management with Kerlink network operations solutions, the Wanesy Management Center.

Thet are also compatible with the Wanesy Small Private Network solution to deploy standalone and self-operating LoRaWAN private network.

They both embed:

  • Ingress protection (IP30)
  • 8ch RX (125 kHz, multi Spreading Factor) + 1ch RX (250KHz or 500kHz, mono Spreading Factor) + 1ch RX (FSK) to get 10ch RX + 1ch TX
  • Built-in high-rejection filters
  • Power supply (230VAC/12VDC) with provided jack connector 2.5mm/5.5mm
  • Highly secured device relying on a hardware secure core (Option)
  • Backhaul connectivity: Ethernet (RJ45)

Wirnet iFemtoCell in the IoT chain

Embedded Base Station Controller and Radio Network Controller, for remote operations

Strengths

 

High performance, reliability & robustness
• Semtech Reference Design v1.5 components,
• Integrated Wi-Fi antenna,
• Built-in with high rejection filters.

 

Security HM and SW architectures
• SecureBoot (Signed firmware),
• SecureStorage (keys and certificates in secured area) using ProvenCore™ solution,
• Secured links and backhaul protection (OpenVPN/IPsec),
• Reboot (watchdog) and recovery on previous Management config (or factory config if the boot issue is not fixed).

Technical features

 

• Sniffer for LBT (Listen Before Talk),
• Rx Sensitivity: -141 dBm (SF12),
• TX Power: configurable from 5dBm to 27dBm,
• Range -20°C +55°C, (for gateway only, without power supply),
• Humidity: 5% to 95%,
• Size: 160 x 90 x 35 mm,
• Weight: 163g (Gateway only); 372g (with packing),
• Spectrum analysis compliant,
• Casing: IP30,
• CPU: ARM Cortex A9,
• DDRAM 256MB,
• 8GB eMMC (6GB available for user).
